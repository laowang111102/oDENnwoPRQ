# 项目概述 python405

本项目是基于Python的图书借阅推荐系统，运用用户协同过滤算法，结合Django框架和MySQL数据库进行开发，前端使用Vue.js构建。系统实现了用户注册、登录、图书搜索、借阅、归还等核心功能，并提供了完善的后台管理模块。

# 技术栈

## 用户-图书评分矩阵

通过`build_user_item_matrix()`函数构建用户-图书评分矩阵，根据用户行为设定评分：

- 收藏书籍：评分1
- 心愿书籍：评分0.5
- 其他书籍：默认评分0

## 用户相似度计算

使用余弦相似度（cosine_similarity）计算用户之间的相似度。

## 推荐逻辑

为目标用户推荐与其行为相似的其他用户所喜欢的图书，推荐数量限制为最多12本。

## 协同过滤

基于用户的协同过滤，通过计算用户之间的相似度，推荐相似用户喜好的图书。
